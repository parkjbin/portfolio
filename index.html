유즈케이스 다이어그램

위 사진은 제가 첫번째로 작성해본 유즈케이스 다이어그램 입니다. 시스템의 흐름도를 보면, 처음 가게를 오픈할시, 지점과 직원을 선택합니다. 그 뒤, 관리자와 구매자(회원)모드로 나뉩니다. 관리자 모드에선 회원 생성,탈퇴,목록조회가 가능하며 회원 모드에선 회원가입, 회원탈퇴가 가능합니다. 기존에 가입해둔 계정이 있다면 그것을 이용해서도 로그인이 가능하고, 로그인 후엔 메인메뉴, 서브메뉴, 음료, 리뷰이벤트의 과정을 거친 뒤 영수증을 출력 받습니다.

  • review: 처음으로 해본 프로젝트라서 그런지 다이어그램 작성과 기능 구현에 많은 어려움을 겪었습니다. 힘들 때 마다 인터넷 검색을 주로 이용 했으며, 다시 책을 보며 자바의 기본 개념에 대해 공부했습니다. 첫 프로젝트 후 스스로 많이 성장하고, 손과 머리에 더욱 자바를 익혔다는 느낌이 들었습니다. 생각했던 내용을 코드로 구상한다는 점이 신기했습니다. 앞으로 프로젝트를 더 많이 진행해보면 실력이 더 많이 늘것이라는 생각을 했습니다. 꿈꾸던 개발자의 모습에 더욱 다가간거 같아서 흐뭇했지만, 그만큼 아쉬움도 많이 남았던 프로젝트 였습니다.

기능구현

프로젝트를 완성 시킨후, 이클립스에서 실행시에 콘솔창에 출력되는 화면을 캡쳐한 것입니다 관리자든, 구매자든 4개의 지점 중 한곳을 선택해야 합니다 .그리고 스텝(에릭)을 한 명 고용합니다. 첫째로 관리자 모드로 접속해 보았습니다. 기존에 저장해둔 관리자 계정을 이용해 로그인 했고, 새로운 회원(유정)을 생성시켰습니다. 그뒤 전체 회원을 조회 해보니 기존회원(제시카)에 이어 유정 또한 조회 됩니다. 이제 관리자 모드를 종료하고 구매자모드로 가보겠습니다. 이번엔 구매자모드에서 새 계정(민영)을 만들고, 로그인 했습니다. 그뒤 메인메뉴로 불고기피자, 서브메뉴는 조각치킨 음료는 환타를 주문했습니다. 그뒤엔 영수증이 출력됩니다. 주문 내역엔 선택한 메뉴들이 출력되고, 현재의 주문 시간이 출력됩니다. 또한 임의로 지정해둔 가격이 모두 합산되어 나왔고, 초기에 설정해둔 직원의 이름이 나왔습니다.

  • review: 기획 단계에서 설정했던 모든 기능을 구현하지 못했습니다. map컬렉션, 배열, local 클래스 등 다양한 자바의 기능들을 사용해보았습니다. 난황을 겪었던 점은 재귀함수를 이용하다가 로직이 꼬여서 고생했었습니다. 같이 교육받았던 분들과 기능과 스킬에 대해 대화해보며, 많은 도움을 주고 받았습니다. 또한 실제 피자 가게에서 사용되는 로직과 기능을 넣으려 했으나, 피자가게 시스템에 대해 잘 모르기에 더욱 기획단계가 힘들 었습니다. 현장에선 고객에게 서비스를 제공할떄, 고객이 원하는 것이 무엇인지 확실히 알아야 하고, 고객과의 커뮤니케이션이 기능 구현 못지 않게 중요하겠다 라는 것을 느꼈습니다.

기능 구현

상단 메뉴바의 Europe, tatue of Liberty, Sphnix, Inca, The great Wall 을 이용하면 아래의 각각 유럽 문화 유산 화면, 자유의 여신상 ,잉카 문명, 만리장성으로 이동할 수 있습니다. a태그의 링크기능 을 이용했고, 작업중 힘든 부분은 인터넷 검색을 많이 이용했습니다. 개발자모드를 이용해 템블릿 화면과 휴대폰 화면을 보며, 반응형이 잘 적용되지 않는지 수시로 체크 하면 작업했습니다. 프론트엔드 개발은 평소 직접 봤던, pc화면과 모바일 화면을 직접 꾸밀 수 있다는 점에서 재미있었습니다.

  • review: 다양한 탬플릿에 대해 알 수 있었던 좋은 계기 였습니다. 여러 웹사이트의 다양한 템플릿을 보며, 소스코드를 분석해보며 프론트앤드에 좀 더 가까워 질 수있는 계기였습니다. 여러 유명 웹사이트의 디자인을 따라 해보기도 했고, 디자인 잘하는 여교육생에게 디자인에 대한 노하우를 묻기도 했습니다.

자유게시판 page

comnet : 먼저 자유게시판 페이지의 메인화면 입니다. 게시물 등륵올 위해 글쓰기 버튼을 눌러 보겠습니다. commnet : 이전의 '모든영화 보기'게시물 과는 달리 '자유게시판'은 관리자 뿐만 아니라, 회원가입된 회원유저 또한 게시물 작성이 가능합니다. comment : 게시물을 작성 해 보겠습니다. 제목, 내용, 작성자 순으로 입력해 줍니다. 하단에 수정하기, 돌아가기, 삭제하기 버튼이 있습니다.
comment : 게시물을 눌렀을 때의 해당 게시물의 제일 하단 부분의 버튼 입니다. 수정하기 버튼은 조회된 게시물을 수정하는 기능, 돌아가기 버튼은 '자유게시판'메인 화면으로 돌아가는 기능, 삭제하기 기능은 조회된 게시물을 삭제하는 기능입니다. comment : 위에서 설명 했던 버튼 중 수정하기 버튼을 눌러 해당 게시물의 내용을 수정 (재밋는거 추천해주세요 -> 재밋는거 추천해주세요 진짜로 진짜로!)해 보겠습니다. comment : 마지막으로 자유게시판 메인 페이지의 검색 기능을 이용해 보겠습니다.
제목을 키워드로 잡고 '눈물바다'라고 검색했더니 해당 게시물이 조회됩니다.
